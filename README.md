
**CyberShield — Cybersecurity Awareness Quiz Website**

An interactive, single-page cybersecurity awareness training website built with HTML, CSS, and vanilla JavaScript. CyberShield features 10 fully interactive lessons covering essential online safety topics — each ending with a timed quiz to test your knowledge.

**Key Features:**
- 10 lessons: passwords, phishing, HTTPS, social engineering, public Wi-Fi, 2FA, social media safety, malware, data privacy, and cyberbullying
- Multiple choice, True/False, and interactive quiz formats
- Live password strength analyzer
- Hover-to-reveal phishing red flags on a simulated email
- Countdown timer on every question
- Score tracking, badge rewards, and confetti results screen
- Dark cyberpunk UI — fully responsive for mobile and desktop
- Single HTML file, no frameworks or dependencies — deployable instantly on GitHub Pages

**Built With:** HTML5 · CSS3 · Vanilla JavaScript
